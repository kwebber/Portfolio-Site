Tickets
Take a ticket to travel the world.
This project was created from a prompt to create a set of unified cards containing only typography -- I wanted to interprete cards into something that is meant to be more tactile. Tickets, have a strong sense of purpose and are an important object that can signify the quality of the experience a person has signed up and paid to do.
